# Project_AusTender
This project has the following elements
1. Prepare an automated workflow using API and Python to extract reports from Austender on
1.1 Contracts tendered out
1.2 Upcoming tender contracts to be tendered by government

2. Create the following insights report
2.1 What types of contracts are tendered to competitors, the amounts tendered by contract type and contract duration.
2.2 Create a pipeline of future contracts to tender for and calculate an estimate tender amount based on historical tender awarded data.
